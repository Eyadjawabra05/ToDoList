# ✅ ToDoList API - Node.js & Express

A RESTful API for managing tasks (To‑Do List) built with Node.js and Express.

## 🚀 Features
- ➕ Add a new task  
- 📄 Get all tasks  
- ✏️ Update a task  
- ❌ Delete a task  
- ✔️ Change task status (Completed / Pending)

## 🛠️ Technologies
- Node.js  
- Express.js  
- MongoDB  
- Mongoose

## 📦 Installation

```bash
git clone https://github.com/Eyadjawabra05/ToDoList.git
cd ToDoList
npm install

## Run

```bash
npm run dev

📌 API Endpoints
GET /api/tasks → Get all tasks
POST /api/tasks → Add a new task
PUT /api/tasks/:id → Update a task
DELETE /api/tasks/:id → Delete a task
